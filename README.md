# Oh My Zsh Custom Configuration

Персональная конфигурация Oh My Zsh для быстрого восстановления настроек после переустановки системы.

## Что включено

- **Тема `gnzh-sergo`**: Кастомная тема с отображением:
  - Текущего времени
  - Пути к директории
  - Активного виртуального окружения Python (venv)
  - Git-ветки и статуса изменений
  - Кода возврата последней команды

- **Конфигурация `config.zsh`**: Настройки плагинов и автоматической активации виртуальных окружений

## Установка

1. Склонируйте репозиторий в `~/.oh-my-zsh/custom`:
   ```bash
   cd ~/.oh-my-zsh/custom
   git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ> .
   ```

2. Добавьте в `~/.zshrc` следующую строку (обычно после `export ZSH="$HOME/.oh-my-zsh"`):
   ```bash
   source $ZSH/custom/config.zsh
   ```

3. Перезагрузите zsh:
   ```bash
   exec zsh
   ```

## Структура

```
.

└── plugins/
    └── example/             # Пример плагина
```

## Особенности

- Автоматическая активация/деактивация Python venv при переходе между директориями
- Чистый промпт без лишних элементов
- Git-статус отображается только внутри git-репозиториев
- Виртуальное окружение отображается только когда активно
